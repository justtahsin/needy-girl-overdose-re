## 🌟 Overview

This repository contains the decompiled `Assembly-CSharp` code of the popular game **Needy Streamer Overload** (originally known as *Needy Girl Overdose*). The purpose of this project is to aid in understanding the inner workings of the game logic, game mechanics, and its data structures for educational and modding purposes.

### ⚠️ Disclaimer
**This project is strictly for educational purposes and reverse engineering research.**
No original game assets (images, audio, etc.) or proprietary binaries are included. Please support the official release by purchasing the game!

## 📁 Repository Structure

- `decompiled/Assembly-CSharp/`: Contains the fully decompiled C# scripts representing the core game logic. 
  - *All original game files, binaries, and assets are ignored.*

## 🛠️ Modding & Research

You can use the source files provided here to:
- Study the event system and dialogue triggers.
- Create custom mods or language patches.
- Understand how Unity handles state in this specific game architecture.
